ABOUT

Have the capability to detect bugs in the code, starting from a semicolon!? Well, TECSTASY is a perfect arena to showcase your skills. BUG HUNTER is an event of TECSTASY 2k18 organized as a part of SUDHEE by IT department of CBIT. Bug Hunter is a debugging event in which you can discover the debugger in you. Don't wait until you have a bug to step through your code. Hunt the bugs in the given time and become the ultimate King or Queen of bug hunters. You can choose your interested language of programming to debug. Solve the set of questions given to you in a time of 20 min
If writing code is an art, debugging is black magic.

Instructions

RULES :
 • We are setting up a platform where each team can select their language(C, C++,Java) of interest as their first step to start the game.
• The second step is that they have to pick up a set from the 10 sets available.
• As soon as they select a set the timer starts and each team is given 20 min of time.
• Each set consists of 6 code snippets where the each snippet consists of 5-6 bugs.
• First 2 snippets are easy, next two are medium and the last two are hard.
• Strict monitoring will be going on as the game starts.
• The sets will be changed based on the requirement.
Finally certificates and prizes will be issued which includes THE KING BUG HUNTER and THE QUEEN BUG HUNTER titles.
